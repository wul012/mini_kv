# mini-kv v123 命令结果归档说明

## 本版范围

本轮按用户明确要求暂时不读取跨项目计划，只做 mini-kv 内部拆分。范围限定为测试结构质量重构：把 `tests/command_tests.cpp` 中冗长的 live `SMOKEJSON` 回执断言拆到独立测试 `tests/smokejson_command_receipt_tests.cpp`。

本版不改生产逻辑、不改 `SMOKEJSON` 字段、不改 fixture 内容、不新增执行能力，也不启动 TCP server。目标是降低 `command_tests.cpp` 文件体量，并让 runtime smoke receipt 的断言归属更清楚。
