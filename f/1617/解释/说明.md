## 06-cli-tcp-smoke.png

命令：

```powershell
.\cmake-build-v1617\minikv_cli.exe --log-level error
.\cmake-build-v1617\minikv_server.exe 6628 127.0.0.1 --log-level error --accept-poll-ms 10
```

CLI 输入覆盖 `SET alpha 1`、`GET alpha`、`EXPIRE alpha 60`、`TTL alpha`、`KEYS`、`CHECKJSON GET alpha`、`QUIT`。TCP smoke 发送 `PING dispatch-smoke`、`SET tcp-key tcp-value`、`GET tcp-key`、`CHECKJSON GET tcp-key`、`QUIT`。

结果：CLI 返回插入、读取、TTL、key list、只读 contract JSON 和 `BYE`；TCP 返回 echo、插入成功、读回值、只读 contract JSON 和 `BYE`。本地 server 验证结束后已停止。

意义：证明 `command_dispatch.cpp` 不只在单元测试里可用，也通过真实 CLI 与 TCP/RESP 入口保持原 command 语义。
