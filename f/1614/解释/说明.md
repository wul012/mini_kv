  - `cmake -S . -B cmake-build-v1614 -G "MinGW Makefiles" -DCMAKE_BUILD_TYPE=Debug`
  - `cmake --build cmake-build-v1614 --config Debug --parallel 4`
- 结果：配置和构建成功，最终到 `[100%] Built target ...`。
- 说明：第一次未显式选择 generator 时，本机 CMake 选择 NMake 但当前环境没有 `nmake`；该失败属于本机生成器选择问题，不是代码失败。随后清理刚创建的 build 目录并按 mini-kv/CLion 实际工具链显式使用 MinGW Makefiles，最终构建通过。截图只保留最终有效证据。
