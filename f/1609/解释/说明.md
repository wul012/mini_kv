## 01-cmake-configure.png

对应命令：

```powershell
$cmake -S . -B cmake-build-v1609 -G 'MinGW Makefiles' -DCMAKE_BUILD_TYPE=Debug
```

结果：CMake 配置和生成通过，构建目录写入 `cmake-build-v1609`。同时核对生成头 `cmake-build-v1609/generated/include/minikv/version.hpp`，确认 `version` 仍为 `0.102.0`，`current_version_path_hint` 仍为 `c/102/`。

意义：证明 v1609 将 archive hint 从 `version.hpp.in` 挪到 CMake 变量后，没有改变 frozen runtime fixture 的外部可见输出。
