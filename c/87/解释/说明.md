# v87 运行调试归档说明

## 本版范围

v87 按 `D:\nodeproj\orderops-node\docs\plans\v215-post-dry-run-adapter-roadmap.md` 推进 mini-kv 侧，只补充 `managed_audit_adapter_non_authoritative_storage_receipt`。它给 Node v217 的 managed audit adapter production-hardening readiness gate 一个只读回执：mini-kv 不是 managed audit store，不做 storage write，不开放 admin/restore 执行，不写 managed audit 状态，也不承担 Java 订单权威状态。

本版没有执行 `LOAD`、`COMPACT`、`SETNXEX`、`RESTORE`，没有连接 Java/Node 运行时，没有读取生产密钥，也没有把 mini-kv 放入 Java 事务一致性链路。
