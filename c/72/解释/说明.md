## 04-real-smoke.png

命令：启动真实 TCP server：

```powershell
cmake-build-v72\minikv_server.exe 6412 127.0.0.1
```

然后用 bundled TCP client 发送：

```text
INFOJSON
CHECKJSON LOAD data/retention-restore.snap
CHECKJSON COMPACT
CHECKJSON SETNXEX restore:retention-token 30 value
STORAGEJSON
HEALTH
GET restore:retention-token
QUIT
```

结果：`INFOJSON` 与 `STORAGEJSON` 返回 `0.72.0`；三条 `CHECKJSON` 只返回只读风险合同；`GET restore:retention-token` 返回 `(nil)`，证明没有执行真实 `SETNXEX` 写入。验证结束后服务端已停止。
