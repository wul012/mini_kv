### 05-runtime-smoke.png

命令：

```powershell
.\cmake-build-v65\minikv_server.exe 6405 127.0.0.1
.\cmake-build-v65\minikv_client.exe 127.0.0.1 6405 5000 --connect-retries 10 --retry-delay-ms 100
```

客户端输入：

```text
INFOJSON
COMMANDSJSON
CHECKJSON LOAD data/bundle.snap
CHECKJSON COMPACT
CHECKJSON SETNXEX bundle:token 30 value
STORAGEJSON
HEALTH
GET bundle:token
QUIT
```

结果要点：

```text
INFOJSON -> version=0.65.0
COMMANDSJSON -> command catalog 可读
CHECKJSON LOAD -> side_effects=["store_replace_from_snapshot"]
CHECKJSON COMPACT -> side_effects=["wal_rewrite_when_enabled"]
CHECKJSON SETNXEX -> write_command=true
STORAGEJSON -> order_authoritative=false
GET bundle:token -> (nil)
smoke assertions passed
```

意义：真实 TCP smoke 只解释 bundle 敏感命令的风险，不执行 `LOAD`、`COMPACT` 或 `SETNXEX`。`GET bundle:token -> (nil)` 证明 token 没被写入。
