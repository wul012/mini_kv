# v88 运行调试归档说明

## 本版范围

v88 按 `D:\nodeproj\orderops-node\docs\plans\v217-post-production-hardening-gate-roadmap.md` 推进 mini-kv 侧，主题是 `command_dispatch_quality_receipt` 和读侧 runtime evidence 命令分发整理。

本版只把 `STATS`、`STATSJSON`、`SMOKEJSON`、`STORAGEJSON`、`HEALTH`、`INFO`、`INFOJSON`、`COMMANDS`、`COMMANDSJSON` 归入 runtime evidence command family，并在 `INFOJSON` / `SMOKEJSON` / release fixture 中暴露只读质量回执。它不是新的执行入口，不是 handler table 迁移，不改变写命令、admin 命令、WAL、snapshot、restore，也不让 mini-kv 成为 managed audit store 或 Java order authority。
