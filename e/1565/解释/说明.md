# mini-kv v1565 命令归档说明

本目录记录 v1546-v1565 command catalog single-source refactor 的关键验证截图。本版目标是把 88 个命令的分发、HELP、COMMANDS/COMMANDSJSON/EXPLAINJSON/CHECKJSON 契约、行编辑补全策略收束到 `command_catalog` 单一来源，并用测试与 smoke 证明这是维护性拆分，不是新增执行入口、router、write、WAL 或 Node/Java 参与。

截图通过 Playwright MCP 生成。由于 MCP 禁止直接打开 `file://`，截图时临时启动了只绑定 `127.0.0.1:8765` 的静态 HTTP 服务来展示临时 HTML 日志页；截图完成后该服务已停止，临时 HTML 和日志会在 cleanup gate 删除。
