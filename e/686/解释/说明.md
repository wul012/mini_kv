# mini-kv v686 command evidence archive

## Scope

v686 starts the controlled read-only signed approval capture preflight chain after v685 signed approval template closeout. The new command is `SHARDROUTEVALUESUPPLYSIGNEDAPPROVALCAPTUREPREFLIGHTJSON`.

This version does not capture a signature, emit or accept an approval grant, submit values, import evidence, store raw signature/secret/endpoint material, install a router, write WAL records, or execute restore/load/compact/service-start/live-read behavior.
