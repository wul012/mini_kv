
# 5. 05-key-cache-refresh-smoke.png

对应流程：

```powershell
创建 client.keys，内容为 stale
启动 minikv_server.exe 6404 127.0.0.1
使用 minikv_client.exe --key-cache-file client.keys 发送：
  SET beta two
  SET alpha one
  KEYS
  QUIT
```

结果：

```text
SMOKE OK
KEYS 输出 key_count=2 keys=alpha beta
client.keys 最终保存：
  alpha
  beta
stale 已被移除
```

意义：

```text
真实 server/client 链路证明：KEYS 响应会替换本地 key cache，并在 --key-cache-file 配置下写回磁盘。
```

后台进程处理：

```text
smoke 结束后已停止本轮启动的 minikv_server.exe。
tmp-v41-smoke-archive 已删除。
```
