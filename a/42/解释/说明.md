# 第四十二版运行调试归档说明

本轮版本目标：

```text
让 metrics file 支持可选 JSONL 导出格式。
```

本轮只做 metrics file 输出格式扩展，不改存储核心、不改 RESP 协议、不改 stdout 默认文本日志，保持版本工作量适中。

# 5. 05-jsonl-metrics-smoke.png

对应流程：

```powershell

  SET alpha one
  STATSJSON
```

```text

```

意义：

```text
真实 server/client 链路证明命令行参数、metrics file writer、JSONL 格式化和周期导出能一起工作。
```
