# 第二十六版开发调试运行归档说明

第二十六版在第二十五版启动期 WAL repair 的基础上，补上 WAL 维护观测能力。核心目标是：让 CLI 和 server 能报告 WAL 文件字节数、物理记录数、当前 live key 数量，并在 WAL 历史明显多于当前数据时自动给出 compact 提示。

本轮主要改动：

- 项目版本更新到 `0.26.0`。
- `WalMaintenanceReport` 新增，用于记录 `bytes`、`records`、`live_keys` 和 `compact_recommended`。
- `WriteAheadLog` 新增 `maintenance_report(const Store&)`。
- `CommandProcessor` 新增 `WALINFO` 命令。
- `WALINFO` 在无 WAL 模式下返回 `ERR WAL not enabled`。
- CLI 启动 WAL 时新增 `WAL stats: ... compact_recommended=...` 输出。
- CLI 在建议 compact 时额外输出 `WAL hint: run COMPACT ...`。
- server 启动 WAL 时新增结构化 `event=wal_stats` 日志。
- server 在建议 compact 时额外输出 `event=wal_compact_hint`。
- `wal_tests` 增加 WALINFO / maintenance_report / compact hint / COMPACT 后提示消失的覆盖。
- `command_tests` 增加 WALINFO 参数错误、无 WAL 错误和 HELP 文本覆盖。
- README 更新为 Version 26，并补充 `WALINFO` 和维护提示说明。

## 05-cli-walinfo-smoke.png

- 输入：
  - 连续 10 次 `SET hot valueN`，制造同一个 key 的长 WAL 历史。
  - `WALINFO`
  - `COMPACT`
  - `WALINFO`
  - `EXIT`
- 结果：
  - 启动时输出 `WAL stats: bytes=0 records=0 live_keys=0 compact_recommended=no`。
  - 第一次 `WALINFO` 输出 `wal_bytes=380 wal_records=10 live_keys=1 compact_recommended=yes`。
  - `COMPACT` 返回 `OK compacted 1`。
  - 第二次 `WALINFO` 输出 `wal_bytes=38 wal_records=1 live_keys=1 compact_recommended=no`。
  - 最终 WAL 文件只有 1 行 WAL2，内容为最后一次写入的 `SET hot value9`。
- 意义：确认 CLI 用户入口可以实时查看 WAL 维护状态，并能看到 compact 前后的建议变化。

## 06-server-walinfo-smoke.png

- 命令：启动真实 `minikv_server.exe <port> 127.0.0.1 <wal_path> --accept-poll-ms 25`。
- WAL 构造：
  - 先用 CLI 连续 10 次 `SET srv valueN`，制造同一个 key 的长 WAL 历史。
- 验证流程：
  - 启动 server 并记录 PID。
  - 等待 `event=tcp_listen`。
  - 检查启动日志包含 `event=wal_stats ... wal_records=10 live_keys=1 compact_recommended=true`。
  - 检查启动日志包含 `event=wal_compact_hint`。
  - 用真实 `minikv_client.exe` 连接 server，执行 `WALINFO`、`COMPACT`、`WALINFO`、`GET srv`、`QUIT`。
  - 检查第一次 `WALINFO` 推荐 compact，`COMPACT` 返回 `OK compacted 1`，第二次 `WALINFO` 不再推荐 compact。
  - 检查 `GET srv` 返回最后一次值 `value9`。
  - 检查最终 WAL 文件只有 1 行 WAL2。
  - 验证结束后停止 server 进程。
- 意义：确认 server 启动入口会输出 WAL 维护日志和 compact hint，TCP 客户端也能通过 `WALINFO` 查看并通过 `COMPACT` 消除提示。
