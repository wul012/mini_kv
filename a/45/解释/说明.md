# 第四十五版运行调试归档说明

本轮版本目标：

```text
按跨项目路线图新增 INFOJSON，让 mini-kv 暴露稳定 JSON 元信息，供后续 Node v54 只读接入。
```

本轮只做 mini-kv 自身的 `INFOJSON`，不做 Node 接入、不做 `COMMANDSJSON`、不改变 `STATSJSON` 语义。

---

# 5. 05-infojson-smoke.png

对应流程：

```powershell
启动 minikv_server.exe 6408 127.0.0.1 --metrics-interval-ms 1000 --max-request-bytes 4096
使用 minikv_client.exe 发送：
  SET alpha one
  INFOJSON
  QUIT
```

结果：

```text
SMOKE OK
INFOJSON 输出包含：
  "version":"0.45.0"
  "protocol":["inline","resp"]
  "live_keys":1
  "wal":{"enabled":false}
  "metrics":{"enabled":true}
  "max_request_bytes":4096
```

意义：

```text
真实 server/client 链路证明 INFOJSON 可以给外部控制面提供稳定元信息。
```
